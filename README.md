# anttux
A very simple open-source operating system.

# How to compile
Use the compile.sh script or Makefile

# Features
- Keyboard Driver
- PC Speaker
- Protected Mode
- GRUB
- Strings (sort of lol)
- Beeps and Boops
- Some more stuff
